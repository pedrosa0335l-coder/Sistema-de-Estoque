# Sistema-de-Estoque
Sistema desenvolvido na  linguagem C para Gerenciamento de estoque

🚀 Funcionalidades
- Cadastro e Edição: Registro de itens (Nome, Qtd, Código) com trava de duplicidade.

- Busca Avançada: Localização de produtos por nome ou código identificador.

- Listagem e Remoção: Visualização completa do inventário e exclusão de itens.

🛠️ Tecnologias
- Linguagem: C

- Memória: Alocação dinâmica e ponteiros

- Estrutura: structs para organização de dados

- Compatibilidade: Cross-platform (Windows e Linux)
